# SocketWire Examples

Collection of example projects that demonstrate how to use [SocketWire](https://github.com/Kabanya/SocketWire).

## Contents
- `socketwire-examples/raylib-examples/` — games and interactive examples using Raylib.
- `socketwire-examples/simple-examples/` — minimal servers/clients, bitstream and echo examples.

## MIPT-networked ports
- `simple-examples/math-duel` ports `MIPT-networked/w1`.
- `simple-examples/packet-stream` ports `MIPT-networked/w3`.
- `raylib-examples/entity-eater` ports `MIPT-networked/w4`.
- `raylib-examples/mipt-w2` ports `MIPT-networked/w2` lobby/game-server/client flow.
- `raylib-examples/mipt-w5` ports `MIPT-networked/w5` prediction/snapshot example.
- `raylib-examples/mipt-w7` ports `MIPT-networked/w7` quantized bandwidth example.
- `raylib-examples/mipt-w10` ports `MIPT-networked/w10` cipher/fuzz example.
- `raylib-examples/earn-gold` is a standalone SocketWire example, not a MIPT port.

## Prerequisites
- C++ compiler with C++23 support
- CMake 3.28 or newer
- Optional: Raylib for raylib-based examples
